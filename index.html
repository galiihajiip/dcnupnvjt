Learning Path Gen AI Engineer

Roadmap terstruktur untuk menjadi Gen AI Engineer profesional

01

Belajar Dasar AI

Memahami konsep fundamental Artificial Intelligence, sejarah, dan penerapannya dalam kehidupan sehari-hari

02

Prompt Engineering untuk Software Developer

Menguasai teknik prompt engineering untuk meningkatkan produktivitas development dengan AI

03

Memulai Pemrograman dengan Python

Belajar dasar-dasar pemrograman Python sebagai fondasi untuk pengembangan AI

04

Belajar Machine Learning untuk Pemula

Memahami konsep Machine Learning dan membangun model ML pertama kamu

Siapa Itu Contributor?

Contributor adalah sebutan untuk semua anggota komunitas ini. Contributor dapat ikut belajar, berkontribusi dalam project, membantu kegiatan, atau berpartisipasi dalam program komunitas sesuai minat masing-masing — tanpa pembagian jalur atau kategori. Semua contributor memiliki kesempatan yang sama untuk berkembang.
